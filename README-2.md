# یارِ امتحان — نسخه مستقل

این نسخه‌ای از ربات هست که بیرون از Claude، روی سیستم خودت یا هر هاستی اجرا می‌شه.

## چرا به سرور نیاز داره؟
کلید API هوش‌مصنوعی نباید هیچ‌وقت داخل کدی که توی مرورگر اجرا می‌شه (فرانت‌اند) قرار بگیره، چون هرکسی می‌تونه بازش کنه و کلیدت رو بدزده. برای همین یه سرور کوچیک (`server.js`) هست که کلید رو مخفی نگه می‌داره و بین مرورگر و Anthropic واسطه می‌شه.

## مرحله ۱: گرفتن کلید API
۱. برو به [console.anthropic.com](https://console.anthropic.com)
۲. حساب بساز و یه API key بگیر (این جدا از اشتراک claude.ai هست و بر اساس مصرف هزینه داره)

## مرحله ۲: نصب و اجرا
```bash
# وارد پوشه پروژه شو
cd exam-helper-standalone

# نصب وابستگی‌ها (به Node.js نسخه ۱۸ به بالا نیاز داری)
npm install

# فایل .env بساز
cp .env.example .env
```
حالا فایل `.env` رو باز کن و کلید API خودت رو جای `sk-ant-xxxx...` بذار.

```bash
# اجرا
npm start
```
حالا برو به `http://localhost:3000` — ربات همون‌جاست.

## مرحله ۳ (اختیاری): میزبانی روی اینترنت
اگه می‌خوای همه بتونن ازش استفاده کنن (نه فقط روی سیستم خودت)، می‌تونی این پوشه رو روی یکی از این سرویس‌ها آپلود کنی:
- **Render.com** (رایگان برای شروع)
- **Railway.app**
- **Vercel** (با کمی تغییر برای serverless)
- یه VPS شخصی

فقط یادت نره متغیر محیطی `ANTHROPIC_API_KEY` رو توی تنظیمات همون سرویس (نه توی کد) وارد کنی.

## نکته مهم درباره هزینه
هر پیام که کاربرها به ربات می‌فرستن از اعتبار API خودت کم می‌شه. اگه قراره عمومی منتشرش کنی، بهتره یه محدودیت تعداد پیام یا نرخ درخواست (rate limit) هم اضافه کنی تا هزینه‌ت کنترل‌شده بمونه.
